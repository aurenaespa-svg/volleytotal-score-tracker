# 🏐 VolleyTotal Score Tracker

Aplicación web local para análisis de partidos de voleibol mediante vídeo, permitiendo registrar cada rally desde el saque y navegar por el marcador.

---

## 🎯 Funcionalidades

- 📹 Carga de vídeo local
- 🎯 Registro del instante de saque
- 🟠 Asignación del equipo ganador del rally
- 🔢 Cálculo automático de marcador y sets
- 🔄 Edición de rallies (insertar, borrar, modificar)
- ⏪ Deshacer / Rehacer
- 🔍 Búsqueda por marcador (ej: Set 3, 15-17)
- 🎛 Controles avanzados de vídeo:
  - velocidad (0.25x → 8x)
  - salto por frame
  - saltos temporales
- 🧠 Deducción automática del turno de saque

- 🔁 Alternancia automática del sacador inicial entre sets ordinarios
- 🎲 Asignación manual del sacador inicial en el set decisivo con nuevo sorteo
- ✍️ Sobrescritura manual del sacador inicial por set para casos especiales
- 💬 Comentarios asociados a cada rally
- 💾 Exportación:
  - JSON (proyecto completo)
  - CSV (tabla de puntos)
- 🎮 Overlay de registro sobre el vídeo
- 🔒 Bloqueo de controles overlay para evitar errores

---

## 🧪 Uso

1. Cargar vídeo local
2. Indicar sacador inicial
3. Durante el visionado:
   - marcar el saque
   - asignar el equipo ganador
4. Navegar por el marcador o editar rallies
5. Guardar o exportar los datos

---

## 💻 Instalación

No requiere instalación.

Puedes:
- abrir `index.html` directamente en el navegador  
o  
- usar `standalone.html` como versión autocontenida
